* banco: scripts de criação de banco e staging area
* config: variáveis de configurações. ex: crontab
* docs: documentos 
* etl: ktr, kjb e scripts de execução
    * filesystem: arquivos de ETL utilizados na solução
        * ods: referentes a staging area
        * dw: referentes a dw
            * job principal
            * dim: referentes a dimensões
            * fato: referentes a fatos
    * repo: arquivos de ETL oriundos de repositório PDI
    * scripts: scripts
* server: arquivos do servidor customizados para a solução - seguir a estrutura de diretório do servidor
